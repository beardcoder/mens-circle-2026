# Favicons & Touch Icons

Bitte füge folgende Icon-Dateien hinzu:

- `favicon-16x16.png` - 16x16 PNG favicon
- `favicon-32x32.png` - 32x32 PNG favicon
- `apple-touch-icon.png` - 180x180 PNG für iOS
- `android-chrome-192x192.png` - 192x192 PNG für Android (optional)
- `android-chrome-512x512.png` - 512x512 PNG für Android (optional)

Diese können mit Tools wie https://realfavicongenerator.net generiert werden.

Alternativ kannst du das Logo aus dem Payload CMS verwenden oder die Laravel public/images verwenden.
